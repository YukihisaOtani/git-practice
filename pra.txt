==============キリンでもわかるgit hub===================
まずは、グローバルにあるデータをローカルに保存しよう！！
git clone [ULR]

内容を変更したらアップロードしよう！！！
コマンドプロンプトでアップロードしたいディレクトリに移動しよう！！
cd [ディレクトリ名]

移動したら実行だ！！
git add [ファイル名]

git commit -m"コメントを付けれるぞ！！"

アップロードする準備は出来た！！
あとは実行のみだ！！
git push origin
